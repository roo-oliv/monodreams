### 3. Clamp YSortDepthBias to layer range
**File**: `MonoDreams/System/Draw/YSortSystem.cs` (line 48)

Wrap the depth calculation in `Math.Clamp(..., minDepth, maxDepth)` so bias can never push depth into an adjacent layer. `Math.Clamp` is already used in the same method (line 44).

```csharp
// Before:
drawComponent.LayerDepth = minDepth + t * (maxDepth - minDepth) + spriteInfo.YSortDepthBias;

// After:
drawComponent.LayerDepth = Math.Clamp(
    minDepth + t * (maxDepth - minDepth) + spriteInfo.YSortDepthBias,
    minDepth,
    maxDepth);
```
